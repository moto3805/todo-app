# ToDoリストアプリ

シンプルで使いやすいToDoリスト管理アプリです。

## 機能

- ✅ タスクの追加・削除
- 📊 進捗管理（0%〜100%）
- 🎯 優先度設定（高・中・低）
- 📅 開始日・終了日の設定
- 💾 ローカルストレージでデータ保存

## 使い方

1. `index.html`をブラウザで開く
2. タスク名を入力
3. 優先度と日付を設定（任意）
4. 「追加」ボタンをクリック
5. 進捗ボタンで進み具合を更新

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## インストール

```bash
# リポジトリをクローン
git clone [リポジトリURL]

# フォルダに移動
cd todo-app

# index.htmlをブラウザで開く
```
